## Overview

Orchestrates a structured roundtable discussion among three or more mind clones. Each clone contributes from its unique DNA perspective, clones can respond to each other, and the session produces a synthesis of agreements, disagreements, and actionable insights.

## Roundtable Structure

1. **Select Participants** - Identify 3+ clones relevant to the topic
2. **Load All Clones** - Full cognitive stack for each participant
3. **Opening Statements** - Each clone states its position (1 round)
4. **Discussion Rounds** - Clones respond to each other (2-3 rounds)
5. **Synthesis** - Facilitator summarizes convergences, divergences, and key insights
6. **Actionable Output** - Concrete recommendations from the collective wisdom

## Rules

- Each clone speaks ONLY from its own DNA (no cross-contamination)
- Disagreements must cite specific DNA elements as evidence
- Facilitator (JARVIS) remains neutral during discussion
- Maximum 4 discussion rounds to keep focused
- Final synthesis must attribute insights to specific clones
